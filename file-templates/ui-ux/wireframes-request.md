# Wireframes Request Template

## Purpose
Request visual mockups/wireframes for UI changes prior to implementation.

## Output location path
`/docs/03-ui-ux/wireframes-request.md`

## Definition of Done
- Scope of visual change is clear.
- Required states/screens are listed.
- Owners and due dates are assigned.
