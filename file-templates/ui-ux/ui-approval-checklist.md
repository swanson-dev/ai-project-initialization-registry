# UI Approval Checklist Template

## Purpose
Capture explicit approval decision for UI changes before implementation.

## Output location path
`/docs/03-ui-ux/ui-approval-checklist.md`

## Definition of Done
- Mockups linked.
- Approver and date captured.
- Approval decision recorded.

## Next Actions
- If approved, proceed to implementation planning.
- If rejected, revise UI spec and wireframes.
