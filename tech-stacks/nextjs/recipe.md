# Tech Stack Recipe: Next.js

## Preferred technology
- Next.js (App Router)
- React
- TypeScript

## Allowed languages
- TypeScript
- JavaScript (limited to tooling/bootstrap where justified)

## Recommended tools
- npm or pnpm
- ESLint
- Prettier
- TypeScript compiler checks

## Forbidden deviations
- Introducing backend/runtime languages outside allowed list without formal decision.
- Adding alternate frontend frameworks that bypass Next.js architecture.
- Placing implementation code outside `/app`.

## Mandatory rule
- ALL code inside `/app`.

## Scope note
- Guardrails only. This recipe does not provide scaffolding.
